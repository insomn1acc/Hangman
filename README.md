Проект на Java 17 "Виселица" в процедурном стиле

При запуске игры игроку представляется выбор начать игру [1] или выйти [2]. Этот выбор так же будет предлагаться после каждого раунда.
Из текстового файла берется случайное слово.
Игрок должен отгадать это слово. При вводе верной буквы - она отображается в маске слова.
Если игрок не угадал - он теряет попытку (всего их 6) и постепенно отрисовывается виселица.
